Configuration Supabase

Entrez vos identifiants Supabase pour activer la base de données en ligne. Les données seront partagées entre tous les appareils. Sans Supabase, le site fonctionne en mode local (localStorage).

Instructions Supabase :
1. Créez un compte sur supabase.com
2. Créez un nouveau projet
3. Dans SQL Editor, exécutez :
CREATE TABLE diplomes (id TEXT PRIMARY KEY, prenom TEXT, nom TEXT, formation TEXT, date TEXT, password TEXT, created_at BIGINT);
CREATE TABLE config (key TEXT PRIMARY KEY, value TEXT);
4. Activez RLS et ajoutez une politique "Allow all" pour les deux tables
5. Copiez l'URL et la clé anon depuis Project Settings → API
